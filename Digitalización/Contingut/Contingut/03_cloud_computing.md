## 3.2 Funcions principals del Cloud

### Processament de dades

| FunciГі | DescripciГі | Serveis |
|--------|------------|---------|
| **CГІmput** | ExecuciГі de programes | EC2, VMs, Containers |
| **Batch** | Processament per lots | AWS Batch, Dataflow |
| **Streaming** | Processament en temps real | Kinesis, Pub/Sub |
| **Serverless** | ExecuciГі sense servidors | Lambda, Cloud Functions |

### Emmagatzematge

| Tipus | CaracterГӯstiques | Serveis |
|-------|------------------|---------|
| **Object Storage** | Objectes (S3-like), escalable | S3, Blob, GCS |
| **Block Storage** | Discs virtuals | EBS, Managed Disks |
| **File Storage** | Sistemes de fitxers compartits | EFS, Azure Files |
| **Archive** | Emmagatzematge fred, baix cost | Glacier, Archive |

### Intercanvi d'informaciГі

| FunciГі | DescripciГі | Serveis |
|--------|------------|---------|
| **APIs** | ExposiciГі de serveis | API Gateway |
| **Missatgeria** | ComunicaciГі asГӯncrona | SQS, Service Bus, Pub/Sub |
| **CDN** | DistribuciГі de contingut | CloudFront, Azure CDN |
| **Networking** | Xarxes virtuals | VPC, Virtual Network |

### ExecuciГі d'aplicacions

| Modalitat | DescripciГі | Serveis |
|-----------|------------|---------|

| **Web Apps** | Aplicacions web | Elastic Beanstalk, App Service |
| **Microserveis** | Arquitectura distribuГҜda | ECS, Cloud Run |

## 3.3 Edge Computing

### DefiniciГі

L'**Edge Computing** Г©s el paradigma que porta el processament de dades prop d'on es generen, reduint latГЁncia i ample de banda cap al cloud.

### Per quГЁ Edge?

| Problema Cloud | SoluciГі Edge |
|----------------|--------------|
| LatГЁncia alta | Processament local (<10ms) |
| Ample de banda | NomГ©s enviar dades rellevants |
| Connectivitat | Funciona offline |
| Privacitat | Dades sensibles no surten |
| Cost | Menys transferГЁncia de dades |
