# 1665. Digitalització Aplicada als Sectors Productius

## Informació del Mòdul

| | |
|---|---|
| **Durada** | 33 hores |
| **Crèdits ECTS** | 3 |
| **Distribució** | 4h/setmana × 8-9 setmanes |

---

## Estructura del Mòdul

### Setmanes 1-2: Teoria (8h)
Contingut teòric necessari per afrontar el projecte:
- Setmana 1: Digitalització, IT/OT, THD
- Setmana 2: Cloud, IA, Dades, Seguretat

### Setmanes 3-8: Projecte (21h)
Desenvolupament del projecte de transformació digital:
- Setmana 3: Selecció de cas i anàlisi inicial
- Setmanes 4-5: Disseny d'arquitectura
- Setmanes 6-7: Prototip i documentació
- Setmana 8: Finalització i preparació defensa

### Setmana 9: Defenses (4h)
Presentació i defensa dels projectes.

## Prerequisits

Aquest mòdul és la culminació de les optatives THD. Cada alumne ja ha cursat una especialització en:

- IoT (Internet of Things)
- Big Data
- Intel·ligència Artificial
- Machine Learning
- Cloud Computing

L'objectiu és obtenir una **visió holística** de totes les THD i saber integrar-les en un projecte real de transformació digital.
