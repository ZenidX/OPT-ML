# Galeria de Casos Empresarials

## Com escollir el teu cas

1. **Llegeix tots els casos** abans de decidir
2. **Considera les THD** que ja coneixes de la teva optativa
3. **Valora la complexitat** segons el temps disponible
4. **Parla amb el teu equip** (si treballes en grup)

## Recomanacions per perfil

### Si vens de IoT
- Casos recomanats: 1, 3, 6, 8, 10
- El teu fort: Sensors, connectivitat, edge computing

### Si vens de Big Data
- Casos recomanats: 2, 4, 5, 9
- El teu fort: Pipelines de dades, analytics, visualització

### Si vens de IA/ML
- Casos recomanats: 2, 3, 5, 10
- El teu fort: Models predictius, visió artificial, NLP

### Si vens de Cloud
- Casos recomanats: 4, 5, 7, 9
- El teu fort: Arquitectura cloud, serveis gestionats, escalabilitat
